# Tipos de estadística

- **Estadística descriptiva**
  - Describe y resume datos
- **Estadística inferencial**
  - Utiliza datos para hacer inferencias sobre una población

# Tipos de datos

- **Datos Numéricos (cuantitativos)**
  - Continuos (medidos)
    - Ejemplos: velocidad de un avión,Tiempo de espera en la cola
  - Discretos (contados)
    - Ejemplos: número mascotas, número de paquetes enviados
- **Datos Categóricos (cuantitativos)**
  - Nominales (sin ordenar)
    - Ejemplos: Casado/Soltero, país de residencia
  - Ordinales (con orden)
    - Ejemplos: calificación de un estudiante, escala de satisfacción

> Identificar el tipo de datos es crucial para aplicar las técnicas estadísticas correctas.
> identificando también que tipo de Grafico usar para representar los datos.
>
> Datos Numéricos -> estadísticas de resumen (mean por ejemplo) y gráficos de dispersion (histogramas, boxplots, etc)
> Datos Categóricos -> estadísticas de resumen (moda por ejemplo) y gráficos de dispersion (barras, tortas, etc)
